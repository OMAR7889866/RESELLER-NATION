# RESELLER NATION

Designer Shoes Reseller Store

## Features

- Dark-themed product grid
- Designer shoes: Balenciaga Tracks, Balenciaga Runners, Bottega Orbit
- Click on any shoe to view details
- Smooth scroll to CashApp payment section
- QR code for $omarvargas922 CashApp payments

## Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## To Add Your Logo and QR Code

1. Replace placeholder images in `src/components/PaymentSection.js`
2. Add your actual CASAPP QR code image
3. Add your CASAPP logo to the header

## Deployment

You can deploy this on:
- Vercel
- Netlify
- GitHub Pages
